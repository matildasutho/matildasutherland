Title:
ASCII Tapestry

The ASCII tapestry is an interactive net artwork inspired by algorithms, and the long standing tradition of woven tapestry. 

As Scott Benesiinaabandan alludes to Gwiizens, the Old Lady and the Octopus Bag Device (2020), the algorithm is in fact in all things. Benesiinaabandan interweaves words about algorithms as we know them in popular culture - as lifeless binary code or AI - with a traditional story of the octopus bag device, an algorithm in its own right. The process of creating an IRL tapestry is an algorithm, a potentially collaborative exercise which, thru algorithm, connects beings. The algorithm is not a cold and lifeless set of rational processes but in fact the collective symbioses thru which we as human beings and all beings and elements within the world coexist.

ASCII Tapestry emulates the process of looming squares, making up the tapestry of nine by nine squares using whichever ASCII symbols the user chooses. Over time the tapestry will grow, as user inputs are added to the tapestry in real time. It is a forever growing mass of squares carefully crafted by anybody with access to the internet and a web browser.
